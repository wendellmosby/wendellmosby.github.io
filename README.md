# WendellMosby.com Starter Website

This is a static website starter for Wendell Mosby’s personal brand site.

## Files
- `index.html` — main website
- `styles.css` — visual design and responsive layout
- `script.js` — mobile menu and footer year
- `mockups/` — PNG mockup concepts

## How to preview locally
Open `index.html` in your browser.

## How to customize
1. Replace the photo placeholder with a real headshot.
2. Replace `hello@wendellmosby.com` in the contact form with your email or a form service endpoint.
3. Add real links to LinkedIn, DJing for Diabetes, books, media, and speaking deck.
4. Deploy to Netlify, Vercel, GitHub Pages, Squarespace developer mode, or any basic web host.

## Suggested domain setup
Buy `wendellmosby.com`, then point the domain to the hosting provider you choose.
